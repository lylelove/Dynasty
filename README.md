dynasty——王朝

这是一个纯 Python/PySide6 搭建的桌面小游戏（原基于 Vue+element）。

## 运行游戏

请确保已安装 Python 环境，然后执行以下命令安装依赖并运行：

```bash
pip install -r requirements.txt
python main.py
```

## 代码结构

原先单体 `main.py` 已按功能拆入 `dynasty/` 包。**入口仍是 `main.py`**，逻辑行为与 V0.18 一致。

| 路径 | 职责 |
|------|------|
| `main.py` | 启动入口 |
| `dynasty/models.py` | 人物模型 `Person` |
| `dynasty/resources.py` | 姓名/爵位/庙谥等词库 |
| `dynasty/state.py` | 一局游戏状态 |
| `dynasty/styles.py` | 古风 QSS 主题 |
| `dynasty/app.py` | 主窗口 UI 与年度循环编排 |
| `dynasty/mixins/` | 命名、爵位、宗室、继承、事件、皇帝、国祚 |

**详细说明文档：** [docs/modules/README.md](docs/modules/README.md)

**提示：** 仅模拟男系宗室（无女性/配偶）。长局性能：低爵不繁衍、裁剪无关已故人物、UI 按 Tab 节流刷新。生育/裁剪参数见 `dynasty/mixins/family.py`（`BREED_RANK_MAX`、`PRUNE_THRESHOLD` 等）。

本版本要点：
- **显眼世系树状图**：皇室宗亲以树状图展示完整父系世系——以开国皇帝为根，按代数向下嵌套。节点含姓名（在位者标★今上）、称号、封国、状态、谥号/庙号/尊号与代数。
- **唐制九等爵**：亲王称「某王」（非明清「某亲王」），郡王/国公/县侯等降等；嫡长承袭，无后则过继侄嗣并改父系。
- **庙号/谥号**：按皇帝**整个在位期间**功绩认定——国运轨迹（登基/峰值/谷值/终局）、在位年数、治国手腕综合评分；开国固定褒谥，恶谥仅明显败坏或亡国时用。
- **继承**：皇位嫡长 → 代位 → 兄终弟及 → 旁支；储君/世子均支持代位。
- **繁衍（仅男系）**：皇帝约 30%、亲王约 7%、郡王约 4%；国公以下不模拟生育。参数见 `family.py`。
