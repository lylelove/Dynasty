dynasty——王朝

这是一个纯 Python/PySide6 搭建的桌面小游戏（原基于 Vue+element）。

## 运行游戏

请确保已安装 Python 环境，然后执行以下命令安装依赖并运行：

```bash
pip install -r requirements.txt
python main.py
```
