**路径：** `dynasty/mixins/titles.py`

**职责：** 封号抽取、在世称号展示、宗室死后谥号、授爵与封爵继承（含过继交接）。

## 3. 授爵与谥号

- 年满 **10 岁**授爵（明制年十岁受封）：亲王/郡王抽封号开府；将军中尉直接授职（无封号）。
- 王世子/王长子不另封，待袭父爵。
- 谥号仅授：太子、亲王、郡王、王世子/王长子；将军中尉无谥。
- 亲王一字谥；郡王例用双字谥（仿明制「某某恭定王」）。

## 4. 承袭（世袭罔替）

- 仅亲王/郡王封国承袭：嫡长（含代位）袭**同封同爵**，永不降等。
- 无后则过继侄嗣（`apply_adoption` 改父系）后承封；再无则除国、封号还池。
- 现任皇帝/储君不得承袭藩爵。
- 将军中尉身故无承袭动作（其子出生时即已定爵）。

子嗣爵位与生育见 `family.py`：`child_rank_for`。
